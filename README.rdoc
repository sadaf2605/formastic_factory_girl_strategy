= NestedAttr
Using this plugin you can use Factory Girl to test your formastic forms like that of Active AQdmin without much hassle.

== Usage
=== Install:
  gem 'nested_attr'
=== Basic Usage

  FactoryGirl.define do
    factory :user,class: User do
      username "sadaf2605"
    end
  end

  require 'nested_attr'
  NestedAttr.nested_attr_for(:user)
  
  # If some of your attributes are not needed
  NestedAttr.nested_attr_for(:user, except=[:post])
